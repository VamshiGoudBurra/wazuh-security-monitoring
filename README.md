# Wazuh SIEM Security Monitoring Project

## Project Overview
This project demonstrates real-time security monitoring using Wazuh SIEM on Kali Linux.

## Features
- Failed authentication monitoring
- Threat hunting dashboard
- File integrity monitoring
- Endpoint security validation

## Tools Used
- Wazuh
- Kali Linux
- Nmap
- Linux Logs

## Steps Performed
1. Installed Wazuh SIEM stack
2. Generated failed login attempts
3. Verified authentication failure alerts
4. Performed endpoint file monitoring
5. Investigated alerts in dashboard

## Results
- Detected 19 authentication failure alerts
- Monitored file integrity changes
- Verified real-time alert generation
